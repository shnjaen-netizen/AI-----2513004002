## 1. 本次作业的目的及整体方向说明
- 本次作业以《寻找中式》为主题，主要目的是通过AI图像生成技术探索和表现中式审美特征。在创作过程中，不仅需要完成至少三张图像的生成，更重要的是记录与大模型的对话、提示词设计以及生成过程。作业强调过程的可追溯性与逻辑性，包括平台、模型、提示词和参考图的使用情况。同时，需要对生成结果进行筛选与评价，说明选择依据，并记录是否进行再次生成或后期优化。整体而言，本作业重点在于展示从“主题理解—图像生成—结果分析—优化调整”的完整过程，而不仅是最终的图像效果。

## 2. 课题一主题
- 中国给人的三种典型印象对比与融合
- 以“中国视觉印象”为核心，提取三种风格元素：
1. 传统水墨画风（古典美学）  
2. 霓虹与新都市风格（强视觉刺激）  
3. 现代SNS风格（柔和情绪表达）  
- 本次重点结合：  传统水墨 × 霓虹都市，突出“静与动”“过去与未来”的对比
